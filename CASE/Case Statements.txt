SYNTAX:

    case expression in 
    CASE1) Command-list;;
    CASE2) Command-list;; .... CASEN)Command-list;;

    esac 

Each case is an expression matching a pattern. The commands in the command-list
for the first match and executed. The | symbol is used for seperating
multiple patterns , and ")" operator terminates a pattern list.
Each case plus it's according commands are called a clause.
Each clause must be terminated with ";;". Each statement  is ended
with the esac statement
